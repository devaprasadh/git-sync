# git-sync
CI plugin to sync the contents of a git repository with another

## Usage
### Docker

```shell
docker run --rm \
  -e PLUGIN_DEBUG=true \
  -e PLUGIN_START_COMMIT=29186cd \
  -e PLUGIN_TARGET_BRANCH=test \
  -v path/to/repo:/repo \
  -w=/repo \
  devatherock/drone-git-sync:latest
```

Please refer [docs](DOCS.md)
